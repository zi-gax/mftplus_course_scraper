نرم افزار رویت، یکی از پیشگامان طراحی و یکی از محصولات شرکت Autodesk در زمینه مدل سازی اطلاعات ساختمان ( Building Information Modeling ) یا همان BIM می باشد. نرم افزار Revit برای معماران و طراحان، مهندسین سازه و ساختمان، مهندسان فنی بخش های مختلف مانند برق، مکانیک و پیمانکاران طراحی عرضه شده است. نرم افزار رویت ( Revit ) به کاربران اجازه می دهد تا یک ساختمان و اجزای آن را به صورت سه بعدی، هوشمند و مبتنی بر مدل سازی پارامتریک طراحی کرده، و به اطلاعات ساختمان از طریق بانک داده های مدل، دسترسی داشته باشند.
