موتور بازی سازی یونیتی ( Unity ) یکی از قویترین موتور های موجود در زمینه ساخت بازی و برنامه های دو بعدی ( 2D ) و سه بعدی ( 3D ) می باشد. این موتور بازی سازی با بهره گیری از زبان برنامه نویسی ( Programming Language ) سی شارپ ( C ) قابلیت ساخت تقریبا هر نوع بازی و برنامه ای را دارد. بعد از ساخت بازی نیز با قابلیت مالتی پلتفرم ( Multiplatform ) بودن، امکان خروجی بازی به تمام سیستم های عامل معتبر دنیا از قبیل ویندوز ( Windows )، اندروید ( Android )، آی او اس ( IOS ) و لینوکس ( Linux ) را دارا می باشد. همچنین در موارد خاص و با طی مراحل تجاری می توانید بازی خود را بر روی کنسول های ایکس باکس ( XBox ) و پلی استیشن ( Play Station ) منتشر نمایید. با خروجی گرفتن به صورت اچ تی ام ال ( HTML5 ) می توان این بازی ها را بر روی پلتفرم های ایتچ ( Itch ) و استیم ( Steam ) نیز منتشر نمود. از دیگر مزیت های یادگیری یونیتی، آشنایی با نحوه ساخت انیمیشن ( Animation ) و جلوه های ویژه ( Visual Effects ) در ساخت بازی در کنار ایجاد محیط کاربری در ساخت بازی دو بعدی و سه بعدی می باشد. به هنرجویان در این دوره آموزشی در خصوص نحوه در آمدزایی ( Monetizing ) از بازی ساخته شده و نمایش تبلیغات ( Advertising ) نیز آموزش های لازم داده می شود.
