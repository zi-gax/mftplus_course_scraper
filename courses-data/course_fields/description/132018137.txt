جاوا پیشرفته که در گذشته تحت مالکیت شرکت Oracle قرار داشت، پس از یک دعوای حقوقی به شرکت Eclipse واگذار شد. به دلیل مسائل مربوط به مالکیت انحصاری، امروزه این فناوری با نام Jakarta EE یا java ee شناخته میشود. گفتنی است که در حال حاضر java ee ، یکی از پرطرفدارترین زبانهای برنامهنویسی است و شرکتهای بسیاری به دنبال استخدام افرادی هستند که دوره آموزش java ee را گذرانده باشند. مراکز آموزشی بسیاری دوره java ee را برگزار میکنند که یکی از معتبرترین مراکز آموزشی در این زمینه مجتمع فنی تهران است.
